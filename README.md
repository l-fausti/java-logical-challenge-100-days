# 🚀 100 Days of Code - Java & Beecrowd

Bem-vindo ao meu repositório do desafio **#100DaysOfCode**!

Este projeto documenta minha jornada de 100 dias focada em consolidar minha lógica de programação em **Java**, resolvendo desafios da plataforma **Beecrowd**.

## 🎯 Objetivo

O objetivo principal é fortalecer os fundamentos de algoritmos e estrutura de dados em Java, preparando a base para avançar em Programação Orientada a Objetos (POO), APIs e Desenvolvimento Web com Spring Boot.

## 📅 Progresso e Organização

Eu me comprometo a resolver pelo menos **2 exercícios por dia** (nas minhas 2 horas disponíveis) e subir a solução aqui.
